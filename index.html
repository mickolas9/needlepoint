About this Project

My concept revolves around creating a dynamic and interactive grid layout inspired by various emotional and physical states or effects, such as nostalgia (neon), void (space), and transformation (prismatic RGB), among others. The art direction focuses on exploring these states through color, light, and interactivity. The design leans heavily into the realms of abstraction and interactivity, where each element on the page is not just seen but experienced. Techniques featured include: CSS Grid for responsive layout, CSS transitions for smooth animations, linear, radial, and conic gradients, 3D transforms, blend modes and filters.
